
# the dashboard for general users



ข้อมูล,รูปแบบการแสดงผล,คำอธิบายสำหรับประชาชน
พื้นที่ระบาด,แผนที่ (Geospatial),"""วันนี้แถวบ้านคุณมีรายงานศัตรูพืชหรือไม่"""
ระดับความน่ากลัว,เข็มไมล์ (Gauge),"""สถานการณ์ภาพรวมในจังหวัดอยู่ในระดับไหน"""
สิ่งที่ต้องระวัง,อันดับ (Top Ranking),"""ศัตรูพืชชนิดไหนที่คุณต้องรู้จักในตอนนี้"""
ทิศทางสถานการณ์,กราฟเส้น (Trend),"""การระบาดกำลังขยายตัวหรือเริ่มสงบลง"""
